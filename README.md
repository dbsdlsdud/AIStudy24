# 머신러닝

시나리오
mbc상점에서 앱마켓을 운영하는데 AI를 활용하는 기법 학습

1. 앱마켓에서 살아있는 생선을 팔기 시작했다.
2. 물류센터에서 생선을 고르는 직원이 있는데 생선이름을 외우지 못했다.
3. 생선의 종류는 도미, 곤들매기, 농어, 강꼬치고기, 빙어, 송어 등등
4. AI미션 : 생선의 길이가 30cm이상이면 도미
